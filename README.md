# Daily Review — Claude Code Skill

Скилл для ежедневного анализа сессий Claude Code. Читает логи за день, находит паттерны и проблемы, предлагает конкретные улучшения конфигурации.

## Что делает

- Собирает все сессионные логи за день
- Анализирует: где буксовали, что сработало, какие ошибки повторялись
- Проверяет здоровье системы (CLAUDE.md, memory, скиллы)
- Предлагает конкретные улучшения с обоснованием
- Создает скомпанованный дневной лог
- Архивирует обработанные сессии
- По пятницам — расширенный аудит всей системы

## Установка

```bash
# Скопировать в папку скиллов
mkdir -p ~/.claude/skills/daily-review
cp SKILL.md ~/.claude/skills/daily-review/
```

## Использование

```
/daily-review
```

Или просто скажи:
- "доброй ночи" — запустится автоматически
- "анализ дня"
- "подведи итоги дня"

## Требования

- Claude Code CLI
- Сессионные логи в `~/.claude/sessions/session-*.md`

## Структура путей

```
~/.claude/sessions/
├── session-2026-03-02-topic.md    # Дневные логи сессий
├── daily/
│   └── daily-2026-03-02.md        # Скомпанованные дневные логи
└── archive/                        # Архив обработанных сессий
```

## Формат вывода

```
## Что шло хорошо
- ...

## Где буксовали
- ...

## Предложения по улучшению
### 1. Добавить правило X в CLAUDE.md
- Проблема: контекст терялся в длинных сессиях
- Решение: автоматический /context каждые 15 шагов
- Где: CLAUDE.md → Принципы работы
```
